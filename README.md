Face Recognition using PCA (Eigenfaces)

📌 Project Overview
This project implements a face recognition system using Principal Component Analysis (PCA). PCA reduces the dimensionality of facial images and extracts key features (Eigenfaces) for efficient recognition.

 🧠 Concept
- PCA converts high-dimensional image data into lower dimensions
- Captures important facial features
- Improves performance and reduces computation

 🛠️ Technologies Used
- Python
- NumPy
- OpenCV
- Scikit-learn
- Matplotlib

 ⚙️ Workflow
1. Load face dataset
2. Preprocess images (grayscale, resize)
3. Apply PCA for dimensionality reduction
4. Train classifier (KNN / SVM)
5. Predict and evaluate results


- Accuracy: 72%
